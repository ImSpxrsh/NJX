# Research to design

Exact references remain to be supplied; this document intentionally does not
invent study findings or citations.

| Research theme                   | Design decision                                           |
| -------------------------------- | --------------------------------------------------------- |
| Weak human voice-clone detection | Do not ask users or software to judge a voice             |
| Urgency and secrecy              | Insert an explicit pause and hold                         |
| Age-related usability needs      | Large targets, one main action, read-aloud, landline path |
| Zero-trust principles            | Do not trust the channel carrying the request             |
| Replay resistance                | One-time, hashed, expiring verification links             |
| Explainable guidance             | Show visible evidence categories and plain reasons        |
| Accessibility exclusions         | Printed card and no-smartphone phone route                |
